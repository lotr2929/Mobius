1. Overview

Mobius is a personal AI assistant deployed as a Progressive Web App (PWA). It routes queries to multiple AI models, integrates with Google services, and can access local files on your device.

Mobius runs in the browser and can be installed on desktop and mobile as a standalone app.

2. Accessing Mobius

  • Web: https://mobius-8e5m.onrender.com
  • Login: Enter your username and password on the login screen.
  • PWA Install: On Chrome/Edge, click the install icon in the address bar to install Mobius as a desktop or mobile app.

3. Interface

The Mobius interface has three main areas:

  • Input area — The text box at the top where you type your query or command.
  • Toolbar — The row of controls below the input area. Contains the + (attach file), Ask button, navigation arrows, and Search box.
  • Response panel — The area below the toolbar where queries and AI responses are displayed.

4. Asking a Question

Mobius uses a two-step process for AI queries:

  • Step 1 — First Ask: Type your question and press Ask. Mobius prepares and displays a query package showing what will be sent to the AI. Review it before sending.
  • Step 2 — Second Ask: Press Ask again to send the query to the AI and receive a response.
Ctrl+Enter can also be used instead of clicking Ask.

5. Choosing an AI Model

By default, Mobius routes queries to Groq. To direct a query to a specific model, use the Ask: prefix:

CommandRoutes to
Ask: GeminiGoogle Gemini 2.5 Flash
Ask: GroqGroq (default)
Ask: MistralMistral Codestral (coding)
Ask: WebsearchTavily web search + Groq

Example: Ask: Gemini. What are the best AI models available today?

The Ask: prefix and model name are stripped before sending — only your actual question reaches the AI.

6. Attaching Files

  • Click the + button in the toolbar to open the file picker.
  • Supported types: images (jpg, png, gif), PDF, text files (txt, md, csv, js, py, html, json).
  • Attached files appear as chips in the toolbar. Click a chip to remove it.
  • If an image is attached, Mobius automatically routes to Gemini for vision processing.

7. Mobius Commands

Colon-prefix commands are handled by Mobius before any AI is involved. They do not go to the AI.

Access:

Grants Mobius access to a folder on your device.

  • Usage: Type Access: and press Ask.
  • A folder picker opens. Navigate to your root folder and click Select Folder.
  • Access is saved between sessions. Run Access: again to change folder.
  • After granting access, folder contents are automatically listed.
  • Browser support: Chrome and Edge on desktop and Android only.

Ask:

Routes your query to a specific AI model.

  • Usage: Ask: modelname. your question
  • Example: Ask: Gemini. What are the best AI models today?
ModelRoutes to
GeminiGoogle Gemini 2.5 Flash
GroqGroq (default)
WebsearchTavily + Groq
The Ask: prefix is stripped before sending to the AI.

Find:

Searches for files and folders by name within the current root folder.

  • Usage: Find: filename
  • Example: Find: server.js
  • Case-insensitive. Returns up to 200 results with full relative paths.
  • If no access granted, automatically runs Access: first.

List:

Lists the contents of the current root folder.

  • Usage: Type List: and press Ask.
  • If no access granted, automatically runs Access: first.

History:

Displays your past conversation sessions from the database.

  • Usage: Type History: and press Ask.
  • Sessions are grouped by time and listed with title and date.
  • Right-click a session for: Open chat, Copy summary, Copy full chat, Copy as context.

8. Chat History

Mobius saves all conversations to a database. To view past sessions, type History: and press Ask. This is a Mobius command — it does not go to the AI.

Right-clicking a session gives four options:

  • Open chat — Loads the full session into the response panel.
  • Copy summary — Copies a brief summary (first 3 exchanges) to the clipboard.
  • Copy full chat — Copies the entire session to the clipboard.
  • Copy as context — Attaches the session summary as a file chip for your next query.

9. Searching the Response Panel

  • Use the Search box in the toolbar to find text within the current response panel.
  • Use the ▲▼ navigation arrows to move between previous and next queries in the current session.

10. Google Integration

Mobius can connect to your Google account to access Drive, Gmail, Calendar, and Tasks. Once connected, Mobius automatically routes relevant queries based on trigger phrases:

ServiceTrigger phrases
Drive"drive", "my files", "google drive"
Tasks"task", "todo", "to-do"
Calendar"calendar", "schedule", "events"
Gmail"email", "gmail", "inbox"

11. Environment Awareness

Mobius automatically detects and sends your environment context with every query. The AI is aware of:

  • Current date, time, timezone and UTC offset
  • Your location (city, region, country) via IP geolocation
  • Local currency
  • Operating system and browser
  • Device type (desktop, tablet, mobile)
  • Network connection type, bandwidth, and latency (where available)
  • Dark/light mode preference
This context is visible in the Mobius Query preview box before sending.

12. Tips

  • Use Ctrl+Enter to trigger Ask without clicking the button.
  • The query preview box shows exactly what Mobius will send — check it before the second Ask.
  • Access: only needs to be run once per browser — permission is remembered.
  • Run Access: again to change to a different root folder.
  • Use Copy as context from chat history to carry a previous session into a new query.
  • Use Find: to locate a file, then attach it manually with the + button.
